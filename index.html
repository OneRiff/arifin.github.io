Apa Itu Website ?

Website adalah kumpulan halaman web yang dapat diakses publik dan saling terkait yang berbagi satu nama domain. Website dapat dibuat dan dikelola oleh individu, grup, bisnis, atau organisasi untuk melayani berbagai tujuan.

HTML adalah bahasa standar pemrogaman yang digunakan untuk membuat halaman website, yang diakses melalui internet. Singkatan dari "Hypertext Markup Language" atau "bahasa markup".

Sir Berners Lee

www & HTML

CSS adalah singkatan dari cascading style sheets, yaitu bahasa yang digunakan untuk menentukan tampilan dan format halaman website. Dengan CSS, Anda bisa mengatur jenis font, warna tulisan, dan latar belakang halaman.

Håkon Wium Lie

CSS

PHP adalah singkatan dari Hypertext Preprocessor, yaitu bahasa pemrograman yang sebenarnya mirip dengan JavaScript dan Python.

Rasmus Lerdorf

PHP

JavaScript adalah bahasa pemrograman yang digunakan untuk pengembangan website agar lebih dinamis. Ibarat kata, JavaScript memberikan “kehidupan” dalam website sehingga terciptanya interaksi antara pengunjung dengan situs tersebut.

Brendan Eich
